wp_insert_category( mixed $catarr, bool $wp_error = false ) : int|object

Updates an existing Category or creates a new Category.

Parameters
Name Type Description
$catarr mixed

See defaults below. Set 'cat_ID' to a non-zero value to update an existing category. The 'taxonomy' key was added in 3.0.0.
$wp_error bool

Optional, since 2.5.0. Set this to true if the caller handles WP_Error return values.
Returns
Type Description
int|object The ID number of the new or updated Category on success. Zero or a WP_Error on failure, depending on param $wp_error.
Details
Since
2.0.0  
wp_update_category( array $catarr ) : int|bool

Aliases wp_insert_category() with minimal args.

If you want to update only some fields of an existing category, call this function with only the new values set inside $catarr.

Parameters
Name Type Description
$catarr array

The 'cat_ID' value is required. All other keys are optional.
Returns
Type Description
int|bool The ID number of the new or updated Category on success. Zero or FALSE on failure.
